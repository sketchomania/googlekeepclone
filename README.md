## Installation

### Clone the repo

```
git clone https://github.com/sketchomania/googlekeepclone.git
```

### Frontend

- change the directory
  ```
  cd client
  ```
- install all dependencies
  ```
  npm install
  ```
- create a .env file
  ```
  touch .env
  ```
- add the followin to ".env" file
  ```bash
  # replace with your
  EFSDF_KEY = DFASDF
  EFSDF_KEY = DFASDF
  ```
- Start the client
  ```
  npm start
  ```
